# graph-mcp-vault

This is a multi-tenant MCP proxy server exposing Neo4j as an MCP tool server, with identity management via a standards-compliant OIDC/OAuth2 provider (Pocket ID is one example).

See **[AGENTS.md](./AGENTS.md)** for project conventions, stack, and development workflow.
See **[DECISIONS.md](./docs/DECISIONS.md)** for all architectural and technical decisions.
See **[PLAN.md](./docs/PLAN.md)** for the full implementation spec.
